# Team Profile Generator

Answer a series of questions to generate a team page.

## Requirements

- Node.js

## Installation

    $ git clone https://github.com/ryanraposo/team-profile-generator
    $ cd team-profile-generator
    $ npm install

## Usage

    $ cd team-profile-generator
    $ node index

## Testing

    $ npm run test
